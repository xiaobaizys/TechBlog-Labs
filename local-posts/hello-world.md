## 为什么写博客

搭建一个技术博客是我一直以来的想法。主要有以下几个原因：

1. **知识沉淀** — 把学习过程中的思考记录下来
2. **技术分享** — 帮助遇到同样问题的开发者
3. **个人品牌** — 建立技术影响力

## 技术栈选择

这个博客使用以下技术栈：

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS + Framer Motion
- **数据库**: PostgreSQL + Prisma
- **认证**: NextAuth.js v5
- **AI**: DeepSeek API
